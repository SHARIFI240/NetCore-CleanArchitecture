# Enterprise ASP.NET Core Web API

A modern and scalable backend project built with ASP.NET Core Web API using Clean Architecture and CQRS principles.  
This project demonstrates enterprise-level software architecture with MediatR, SQL Server, JWT Authentication, and clean separation of concerns.

## 🚀 Technologies

- ASP.NET Core Web API
- SQL Server
- Entity Framework Core
- Clean Architecture
- CQRS
- MediatR
- JWT Authentication
- Dependency Injection

## ✨ Features

- Clean and maintainable architecture
- JWT-based Authentication & Authorization
- CQRS implementation using MediatR
- RESTful API design
- Scalable and testable structure
- SQL Server integration

## 📂 Architecture Layers

- Domain
- Application
- Infrastructure
- Presentation/API

## 🎯 Purpose

This repository was created as a sample project for developers interested in learning and implementing modern backend architecture patterns in ASP.NET Core.
